# Facebook Flyer/Advertisement Guide for MixedMakerShop

## 📐 Recommended Sizes

### Facebook Link Preview (Best for sharing)
- **Size**: 1200 x 628 pixels
- **Format**: JPG or PNG
- **File size**: Under 1MB
- **Use**: When sharing a link to your website

### Facebook Post (Square)
- **Size**: 1080 x 1080 pixels
- **Format**: JPG or PNG
- **File size**: Under 1MB
- **Use**: For regular posts in feed

### Facebook Story
- **Size**: 1080 x 1920 pixels (9:16 ratio)
- **Format**: JPG or PNG
- **Use**: For stories

## 🎨 Design Files Created

1. **facebook-flyer.html** - HTML file you can:
   - Open in browser
   - Take a screenshot (use browser dev tools to set exact dimensions)
   - Convert to image using online tools

## 📝 How to Create the Image

### Option 1: Screenshot Method
1. Open `facebook-flyer.html` in your browser
2. Press F12 to open Developer Tools
3. Click the device toolbar icon (or press Ctrl+Shift+M)
4. Set custom dimensions: 1200 x 628
5. Take a screenshot (Windows: Win+Shift+S, Mac: Cmd+Shift+4)
6. Crop to exact dimensions

### Option 2: Online Tools
1. Open `facebook-flyer.html` in browser
2. Use tools like:
   - [htmlcsstoimage.com](https://htmlcsstoimage.com)
   - [screenshot.guru](https://screenshot.guru)
   - [html2canvas](https://html2canvas.hertzen.com)

## 📸 Tips for Best Results

1. **Use the HTML file** - It's already designed with your brand colors
2. **Test on mobile** - Make sure text is readable on small screens
3. **Include your URL** - Always include mixedmakershop.com
4. **Show examples** - Mention GoneFishin & Kelsey'sKustom as proof
5. **Clear CTA** - "Visit MixedMakerShop" or "Get a Quote"
6. **Post timing** - Best times: 9am, 1pm, 7pm (your timezone)
