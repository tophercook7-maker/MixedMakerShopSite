START HERE (READ THIS FIRST)
============================

You only edit TWO files:
1) EDIT_BASICS.txt  (name, phone, address, hours)
2) EDIT_MENU.txt    (menu items and prices)

Use:
- Windows: Notepad
- Mac: TextEdit

STEP 1 — Edit the basics
------------------------
Open EDIT_BASICS.txt, change the info, save.

STEP 2 — Edit the menu
----------------------
Open EDIT_MENU.txt, change menu items/prices, save.

STEP 3 — Put your info into the website
---------------------------------------
Open index.html in Notepad/TextEdit.
Use Find (Ctrl+F / Cmd+F) and replace these placeholders:

__BUSINESS_NAME__
__TAGLINE__
__PHONE__
__EMAIL__
__ADDRESS1__
__ADDRESS2__
__HOURS__

For the menu:
Find __MENU_TEXT__ and replace it with the contents from EDIT_MENU.txt.

STEP 4 — Preview
----------------
Double-click index.html to preview.

STEP 5 — Publish on Netlify
---------------------------
1) Go to netlify.com and create a free account
2) Drag-and-drop this entire folder
3) Your website is live
